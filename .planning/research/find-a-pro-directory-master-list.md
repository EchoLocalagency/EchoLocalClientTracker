## Pipeline Automation Plan

### Phase 1: One-Time Setup (per client)
1. Identify which trade the client is in
2. Pull the relevant manufacturer directories from Tier 1
3. Pull all Tier 2 trade associations for their state
4. Compile Tier 3 directories (same list for all clients)
5. Check which ones the client is already listed on (search business name + phone)

### Phase 2: Submit (per client)
- **Form-fill directories (Tier 3):** Can batch-submit 10-15/day. Templated info, just swap client details.
- **Manufacturer directories (Tier 1):** Need client input (certifications, products used, etc.). Email or form per manufacturer.
- **Trade associations (Tier 2):** Client must join/pay. We identify and recommend, client signs up.

### Phase 3: Track
- Spreadsheet per client: Directory name, submission date, status (pending/approved/rejected), live URL, DA
- Follow up on pending submissions after 7 days
- Verify links are indexed via Google `site:` search after 2-4 weeks

### Expected Output Per Client (Month 1)
- 15-25 new directory listings from Tier 3 (form fills)
- 2-5 manufacturer directory applications from Tier 1
- 1-2 trade association recommendations from Tier 2
- Total new backlinks: 20-30
