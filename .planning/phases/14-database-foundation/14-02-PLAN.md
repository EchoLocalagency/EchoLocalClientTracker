**Part A: Add `create_pipeline_lead_from_call()` function to analyze_calls.py**

Add the function before `store_analysis()`. It should:

1. Check if `analysis.get("outcome")` is in `("meeting_booked", "closed")`. If not, return immediately.
2. Accept analysis_id as a parameter (passed from store_analysis after insert).
3. Extract phone from `call.get("call_from") or call.get("phone")` and company_name from `call.get("company_name")`.
4. Duplicate detection:
   - If phone is not None: query `pipeline_leads` where phone = phone, limit 1. If exists, print skip message and return.
   - If company_name is not None: query `pipeline_leads` using `.ilike("company_name", company_name)`, limit 1. If exists, print skip message and return.
5. Build lead dict: contact_name (from `call.get("contact_name") or "Unknown"`), phone, company_name, source="sales_engine", stage="Lead", notes (from `analysis.get("caller_details", {}).get("situation", "")`), call_analysis_id=analysis_id.
6. Insert into `pipeline_leads`. On success, also insert initial stage history row into `pipeline_stage_history` with previous_stage=None, new_stage="Lead", notes=f"Auto-created from call (outcome: {outcome})".
7. Print success/failure messages prefixed with `[pipeline]`.
