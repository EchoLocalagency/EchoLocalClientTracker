**Table 1: pipeline_leads**
- id uuid DEFAULT gen_random_uuid() PRIMARY KEY
- contact_name text NOT NULL
- email text (nullable)
- phone text (nullable)
- company_name text (nullable)
- trade text (nullable)
- source text NOT NULL DEFAULT 'manual'

- notes text (nullable)

- created_at timestamptz NOT NULL DEFAULT now()
- updated_at timestamptz NOT NULL DEFAULT now()

**Table 2: pipeline_stage_history**
- id uuid DEFAULT gen_random_uuid() PRIMARY KEY
- lead_id uuid NOT NULL REFERENCES pipeline_leads(id) ON DELETE CASCADE
- previous_stage text (nullable -- NULL for initial entry)
- new_stage text NOT NULL
- transitioned_at timestamptz NOT NULL DEFAULT now()
- notes text (nullable)
- Index: lead_id

**Table 3: pipeline_checklist_items**
- id uuid DEFAULT gen_random_uuid() PRIMARY KEY
- lead_id uuid NOT NULL REFERENCES pipeline_leads(id) ON DELETE CASCADE
- stage text NOT NULL
- item_key text NOT NULL
- item_label text NOT NULL
- completed boolean NOT NULL DEFAULT false
- completed_at timestamptz (nullable)
- created_at timestamptz NOT NULL DEFAULT now()
- Index: lead_id, (lead_id + stage) composite
- UNIQUE(lead_id, stage, item_key) -- prevent duplicate checklist items

**Table 4: pipeline_comms**
- id uuid DEFAULT gen_random_uuid() PRIMARY KEY
- lead_id uuid NOT NULL REFERENCES pipeline_leads(id) ON DELETE CASCADE
- comm_type text NOT NULL -- 'call', 'email', 'text'
- direction text NOT NULL DEFAULT 'outbound' -- 'outbound', 'inbound'
- notes text (nullable)
- occurred_at timestamptz NOT NULL DEFAULT now()
- created_at timestamptz NOT NULL DEFAULT now()
- Index: lead_id, occurred_at

**RLS for ALL four tables:** Enable RLS, create admin-only policy using user_profiles check pattern:
```sql
ALTER TABLE {table} ENABLE ROW LEVEL SECURITY;
CREATE POLICY "{table}_admin_only"
    ON {table}
    FOR ALL
    TO authenticated
    USING (
        EXISTS (
            SELECT 1 FROM user_profiles
            WHERE user_profiles.user_id = auth.uid()
            AND user_profiles.role = 'admin'
        )
    )
    WITH CHECK (
        EXISTS (
            SELECT 1 FROM user_profiles
            WHERE user_profiles.user_id = auth.uid()
            AND user_profiles.role = 'admin'
        )
    );
```
