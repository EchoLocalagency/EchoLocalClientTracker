Create `src/components/pipeline/LeadDrawer.tsx` ('use client'):
- Props: `{ leadId: string | null; onClose: () => void; onLeadUpdated: (lead: PipelineLead) => void }`.
- When `leadId` is null, return null (drawer is closed).
- Render via `ReactDOM.createPortal` into `document.body`:
  - Backdrop div (fixed, inset 0, rgba(0,0,0,0.5), z-index 100, onClick calls onClose).
  - Panel div (fixed, top 0, right 0, bottom 0, width `min(560px, 100vw)`, background var(--bg-surface), borderLeft 1px solid var(--border), z-index 101, overflowY auto, padding 24px 28px, role="dialog", aria-modal="true").
- useEffect on `leadId` change: fetch all 4 tables in parallel via `Promise.all`:
  - `supabase.from('pipeline_leads').select('*').eq('id', leadId).single()`
  - `supabase.from('pipeline_stage_history').select('*').eq('lead_id', leadId).order('transitioned_at', { ascending: true })`
  - `supabase.from('pipeline_checklist_items').select('*').eq('lead_id', leadId)`
  - `supabase.from('pipeline_comms').select('*').eq('lead_id', leadId).order('occurred_at', { ascending: false })`
- State: `lead`, `history`, `checklistItems`, `comms`, `loading` (boolean).
- useEffect for Escape key: add keydown listener when open, remove on cleanup.
- useEffect for body scroll lock: set `document.body.style.overflow = 'hidden'` when open, restore on cleanup.
- Show "Loading..." centered in panel while fetching.
- Render four sections in order:
  1. Header: lead contact_name + company_name, close button (X) top right.
  2. `<LeadProfile>` passing lead + setLead + onLeadUpdated.
  3. `<StageTimeline>` passing history.
  4. Placeholder divs for Checklist and CommsLog (labeled "Checklist" and "Communications" with section headings) -- these will be filled in plan 16-02.
- When `setLead` updates local lead state, also call `onLeadUpdated(updatedLead)` so the pipeline table stays in sync.

Create `src/components/pipeline/LeadProfile.tsx` ('use client'):
- Props: `{ lead: PipelineLead; onFieldSaved: (updated: PipelineLead) => void }`.
- Display fields: contact_name, email, phone, company_name, trade, source, notes.

- State: `editing: keyof PipelineLead | null`, `draft: string`.
