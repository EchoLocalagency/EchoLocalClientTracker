1. Create `src/components/pipeline/PipelineAnalytics.tsx` as a 'use client' component accepting props:
   - `leads: PipelineLead[]`
   - `stageHistory: PipelineStageHistory[]`

2. Inside the component, compute three derived datasets via `useMemo`:

   **Funnel data (ANAL-01):** Count unique leads that ever entered each stage by scanning `stageHistory` for `new_stage` values. Seed 'Lead' stage with all lead IDs (every lead starts there). For each stage, compute drop-off percentage vs previous stage. Return `{ stage, count, dropoff }[]` in PIPELINE_STAGES order.

   **Avg days per stage (ANAL-02):** Group history entries by lead_id (already sorted asc by transitioned_at). For consecutive entries, duration = next entry's transitioned_at minus current entry's transitioned_at. For the last entry per lead, use Date.now() as exit time. For leads with NO history entries, compute Lead stage duration from `lead.stage_entered_at` to now. Return `{ stage, avgDays: number | null, sampleSize: number }[]`.

   **Source breakdown (ANAL-03):** Group leads by `source` field. Count per source. Sort descending by count. Return `{ source, count }[]`.

3. Render the component with this layout:
   - Section heading "Analytics" (fontSize 15, fontWeight 600, color var(--text-primary))
   - Row of 6 avg-days metric cards (flex, gap 8, flexWrap wrap) -- each card shows stage name (uppercase mono 10px), avg days value (22px bold mono accent color, format as "X.Xd" or "--"), and sample size below
   - Two-column flex layout (gap 16, flexWrap wrap, minWidth 300 per child):
     - Left: Conversion Funnel -- horizontal BarChart (layout="vertical") with stages on Y axis, lead counts on X axis. Use `Cell` with decreasing opacity per stage (`rgba(6, 182, 212, ${1 - index * 0.12})`). Use ChartTooltip. Height 220px.
     - Right: Lead Sources -- horizontal BarChart (layout="vertical") with source names on Y axis, counts on X axis. Single fill color var(--accent). Use ChartTooltip. Height 220px.
   - Both charts wrapped in card containers (bg-surface, border, radius-card, padding 24) with title and subtitle

4. Follow project styling: inline `style={{}}` only (no Tailwind), CSS custom properties (--bg-surface, --border, --accent, --text-primary, --text-secondary, --font-mono, --radius-card, --danger). Wrap each ResponsiveContainer in a div with explicit `width: '100%'`.
